### **简介** ###

> **项目简介**

软件工程大作业，一款面向大学生群体的二手交易平台App。

> **开发环境**

IDE:Android Studio

Language: Java

> **工程结构**

    main->AndroidManifest.xml 注册文件
	      res                 存放layout和图片等媒体文件的位置
          java                Java代码所处的地方
	APP                       示例APP
    project                   完整的工程文件
